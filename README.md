## Getting Started

Install dependencies,

```bash
npm i
# or
yarn
```

Start the dev server,

```bash
npm start
# or
yarn start
```

Build documentation,

```bash
npm run docs:build
# or
yarn docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run build
```
